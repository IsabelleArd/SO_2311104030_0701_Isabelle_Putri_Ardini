## Dasar Teori

Modul 02 membahas tentang proses instalasi dan konfigurasi environment sistem operasi Xinu. Xinu sendiri merupakan sistem operasi yang dikembangkan dengan konsep cross-platform, dimana sistem operasi tersebut dikembangkan pada satu komputer dan dijalankan pada komputer lain. Pada praktikum ini, kami menggunakan dua mesin virtual, yaitu 'development-system' dan 'backend'. 

'development-system' digunakan sebagai environment pengembangan dan kompilasi Xinu dimana mesin ini berisi source code Xinu, compiler, serta layanan jaringan DHCP dan TFCP. Sedangkan 'backend' digunakan untuk menjalankan 'development-system' dengan melakukan booting menggunakan PXE untuk mengambil file sistem dari 'development-system'. 

## Guided

Pada sesi praktikum Modul 01, dilakukan beberapa langkah berikut:
1. Mengekstrak file xinu-vbox-appliances.tar.gz sehingga diperoleh file development-system.ova dan backend.ova.
2. Mengimpor development-system.ova ke Oracle VM VirtualBox dan melakukan pengaturan jaringan, serial port, dan display sesuai modul.
3. Mengimpor backend.ova dan melakukan pengaturan jaringan serta serial port agar terhubung dengan development-system.
4. Menjalankan development-system dan backend untuk memastikan backend dapat melakukan booting melalui jaringan.
